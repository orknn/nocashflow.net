renamed to English. The Turkish (`/tr/…`) URLs were **not** touched.

GitHub Pages cannot emit native 301s and the zone is behind Cloudflare, so these
redirects must be configured in **Cloudflare → Bulk Redirects** (or Redirect
Rules). All should be **301 (permanent)**, query string preserved.

## Notes
- `bulletin_page.html` was intentionally left unchanged (not a Turkish slug; TR
  side uses the same name).

  before the next deploy to avoid a 404 window.
